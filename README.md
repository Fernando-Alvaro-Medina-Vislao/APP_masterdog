# 🐶 MasterDogApp - Aplicación de Gestión y Cuidado de Mascotas

## 📌 Descripción del Proyecto

MasterDogApp es una aplicación móvil orientada a la gestión y cuidado de mascotas, diseñada para ayudar a los usuarios a organizar información importante sobre sus perros, como salud, alimentación, rutinas y seguimiento general.

---

## 🖥️ Vista de la Aplicación

* Pantalla principal
* Perfil de mascota
* Registro de actividades
* Control de salud
* Recordatorios

---

## 🎯 Funcionalidades Principales

* 🐾 Registro de mascotas
* 📋 Gestión de información (edad, raza, peso, etc.)
* 💉 Control de vacunas y salud
* 🍖 Seguimiento de alimentación
* ⏰ Recordatorios (vacunas, citas, alimentación)
* 📊 Historial de actividades

---

## 📊 Componentes del Sistema

* Gestión de usuarios
